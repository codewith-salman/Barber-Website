## About The Project

The **Barbar Hair Salon Website** is designed to provide a professional online presence for hair salons and barbershops.  
It showcases services, pricing, team members, and customer testimonials while maintaining an elegant and modern style.  

This project is fully responsive, ensuring that customers can explore services and book appointments seamlessly from any device.  
Perfect for small salons, personal barbers, and beauty professionals who want to build credibility and attract new clients online.  

### Built With

- HTML5  
- CSS3 

### Features

- Elegant and modern salon-themed design  
- Fully responsive layout for all screen sizes  
- Services section to highlight offerings  
- Pricing section for easy cost overview  
- Gallery to showcase hairstyles and salon ambiance  
- Barber/Stylist team introduction section  
- Contact form with map and social media integration
